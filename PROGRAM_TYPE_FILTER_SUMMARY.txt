🔄 HOW IT WORKS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. User selects a Program Type from dropdown
  2. Frontend sends program_type parameter to backend
  3. Backend filters ProgramRegistry by program_name field
  4. Returns only programs matching the selected type
  5. Business dropdown automatically updates to show only businesses with that type
