🔄 FLOW CREATE PROGRAM:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Заповнити форму Create Program
2. Натиснути "Create Program"
3. JobTracker з'являється зверху форми
4. Показує статус: PENDING → IN_PROGRESS → COMPLETED
5. При COMPLETED: чекає 1.5 сек → редіректить на program-status
6. Користувач бачить створену програму!

⚙️ ТЕХНІЧНІ ДЕТАЛІ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Polling:
  • RTK Query useGetJobStatusQuery з pollingInterval: 2000ms
  • Автоматично зупиняється після COMPLETED/FAILED

Редірект:
  • setTimeout 1500ms після COMPLETED
  • navigate(`/program-status/${program_id}`)
  • Program ID береться з business_results[0].program_id

API:
  • GET /api/reseller/status/{job_id}
  • Response: { status, created_at, completed_at, business_results }
